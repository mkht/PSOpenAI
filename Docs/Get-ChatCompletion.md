# Get-ChatCompletion

## SYNOPSIS
Get stored chat completions.

## SYNTAX

### Get
```
Get-ChatCompletion
    [-CompletionId] <String>
    [-TimeoutSec <Int32>]
    [-MaxRetryCount <Int32>]
    [-ApiBase <Uri>]
    [-ApiKey <SecureString>]
    [-Organization <String>]
    [<CommonParameters>]
```

### List
```
Get-ChatCompletion
    [-All]
    [-Limit <Int32>]
    [-Order <String>]
    [-TimeoutSec <Int32>]
    [-MaxRetryCount <Int32>]
    [-ApiBase <Uri>]
    [-ApiKey <SecureString>]
    [-Organization <String>]
    [<CommonParameters>]
```

## DESCRIPTION
Retrieves stored chat completions. Only chat completions that have been stored with the store parameter set to true will be returned.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-ChatCompletion -CompletionId "chatcompl-abcd123"
```

Get a completion with the specified ID.

### Example 2
```powershell
PS C:\> Get-ChatCompletion -All
```

Lists all stored completions.
