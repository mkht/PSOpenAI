# Remove-Video

## SYNOPSIS
Deletes a video generation job.

## SYNTAX

```
Remove-Video
    [-VideoId] <String>
    [-TimeoutSec <Int32>]
    [-MaxRetryCount <Int32>]
    [-ApiBase <Uri>]
    [-ApiKey <SecureString>]
    [-Organization <String>]
    [<CommonParameters>]
```

## DESCRIPTION
Deletes a video job that was previously created. Use this to clean up jobs that you no longer need.

## EXAMPLES

### Example 1
```powershell
PS C:\> Remove-Video -VideoId 'video_68ea'
```

Deletes the specified video job.
