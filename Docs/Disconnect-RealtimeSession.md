# Disconnect-RealtimeSession

## SYNOPSIS
Close realtime session.

## SYNTAX

```
Disconnect-RealtimeSession
```

## DESCRIPTION
Terminates a connected conversation session. If audio input/output is activated, they are also terminated.

## EXAMPLES

### Example 1
```powershell
PS C:\> Disconnect-RealtimeSession
```
